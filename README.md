# IQ Mix — Test complet (FR/AR) (GitHub Pages)

Jeu web futuriste (bilingue Français/Arabe) avec 6 modules :
- Logique
- Spatial
- Mémoire de travail
- Calcul mental
- Raisonnement verbal
- Attention (épreuve continue)

> ⚠️ Indicateur ludique, pas un test médical.

## ✅ Lancer en local
Ouvre `index.html` dans ton navigateur.

## ✅ Publier sur GitHub Pages
1. Crée un repo (ex: `iq-mix`)
2. Ajoute tous les fichiers (racine + dossier `assets/`)
3. Dans GitHub : **Settings → Pages**
4. **Build and deployment** : *Deploy from a branch*
5. Branch: `main` / folder: `/ (root)`
6. Attends la publication, puis ouvre l’URL de Pages.

## Personnalisation rapide
- `assets/questions.js` : banque de questions FR/AR
- `assets/styles.css` : look futuriste
- `assets/i18n.js` : textes UI FR/AR
